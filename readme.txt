Run Part A - Process activity with
./run_experiment_A.py

Run Part A - Context Switches with
./run_experiment_A_context_2.py to generate a graph with 2 processes.
./run_experiment_A_context_2.py to generate a graph with 3 processes.

Run Part B - NUMA with
./run_experiment_B.py

Note: The programs may take some time to run (< 1 minute).

Note: You need the tsc.c and tsc.h files in the same folder.

Note: Depending on your python installation you may need to run
python ./run_experiment_*

Note: If your editor adds line cartridges (^M) then you need to run
dos2unix <filename> before running the program.
